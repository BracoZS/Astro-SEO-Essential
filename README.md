# Astro SEO Essential

A simple Astro component to handle basic SEO meta tags in your pages.

## What this component does

- Centralizes your SEO configuration.
- Renders common meta tags:
  - `<title>` and `<description>`.
  - Open Graph tags (for social link previews).
  - Twitter Card tags.

## Typical usage

You import the SEO component into your Astro page or layout and pass the SEO props you need.
